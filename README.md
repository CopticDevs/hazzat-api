# hazzat-api
![logo](http://www.hazzat.com/portals/_default/skins/hazzat/images/logo.png "Hazzat Logo")

Hazzat backend REST API writen in vanilla Node.js with a React frontend. [www.hazzat.com](http://hazzat.com)

## Prerequisites

* Node.js & npm
* SQL Server

## Build

> * `npm install`
> * `npm run build`

## Run

> `npm run start`
